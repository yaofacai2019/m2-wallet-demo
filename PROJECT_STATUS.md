# M2 Wallet 项目状态

更新日期：2026-07-23

## 目标

在两周开发范围内完成内部可运行的稳定币收付 Demo：

- TRON USDT 收款
- Polygon USDC 收款
- 平台佣金计算与记账
- 客户提现申请
- 财务人工确认
- 签名服务接口
- 交易广播、确认与状态回写
- 回调、对账和异常处理
