# Cregis 功能盘点与 M2 Wallet 对照

更新日期：2026-07-22

本文件来自 Cregis 登录桌面端的一次只读浏览，并结合其公开开发文档整理。盘点过程未查看
API Secret、助记词或私钥，未修改设置，未创建钱包，也未发起任何交易。

## 产品信息架构

### 1. 钱包

Cregis 当前桌面端展示的能力：

- 多钱包列表与按钱包查看总资产
- 单签/多签类型与密钥分片标识
- 币种资产、可用金额、处理中金额和可归集金额
- 发送、接收、地址管理
- 按币种查看地址，支持地址别名、地址类型和地址搜索
- 钱包内交易记录与交易详情
- 多链资产展示

M2 Wallet 对照：

- Demo 已有 TRON USDT、Polygon USDC 资产视图
- 首期补充钱包表、地址表、余额状态、Gas/能量状态和归集任务
- 首期不在应用进程中保存私钥；签名始终通过独立签名服务接口

### 2. 协作

Cregis 当前桌面端展示的能力：

- 待办、已办
- 待审批、待签名、已审批、已签名
- 全部事项与“我发起的”筛选
- 按发送方、接收方和日期检索
- 自动签名能力入口

M2 Wallet 对照：

- Demo 已实现财务审批及签名、广播、确认状态链
- 首期补充“待审批/待签名/已完成”工作台与操作人筛选
- 后续增加多人复核、金额阈值和发起人不能自审规则

### 3. WaaS 项目

Cregis 当前桌面端展示的能力：

- 多项目切换
- 钱包提币记录
- 异常回调与历史回调
- 开发者设置、API 文档、SDK 下载
- API Key 与 Base URL（盘点过程中未展开密钥）
- IP 白名单分组
- 回调地址批量设置
- 最小回调金额，用于过滤小额骚扰/粉尘交易
- 提币外部验证、BTC 地址类型
- 异常回调、项目启停、API Key 重置通知
- 生成子地址与 API 提币接口类型
- 提币策略与默认出款钱包绑定

M2 Wallet 对照：

- Demo 已有项目级收款、提现 API、回调事件和失败重试
- 首期补充异常/历史回调页面、手动重试、项目启停、IP 白名单和最小入账金额
- API Key 只显示末四位，重置操作必须二次确认并记录审计日志

### 4. 支付引擎

Cregis 当前账号尚未创建支付引擎项目，因此桌面端只展示创建入口和产品介绍。结合公开文档可确认：

- 多链、多币种支付
- 托管收银台与支付 API
- 订单有效期、支付地址和二维码
- 支付不足、超额支付、补款、过期和退款状态
- 自动结算到指定钱包
- 订单、支付、结算记录与回调
- 商户名称、Logo 和收银台样式

M2 Wallet 对照：

- Demo 已有 USDT/USDC 收款订单、地址、确认和佣金
- 首期补充订单过期、少付/多付、支付页面与二维码
- 商业版本再增加汇率锁定、退款、自动结算及商户化收银台

### 5. 全局交易记录

Cregis 当前桌面端展示的字段和筛选维度：

- 交易时间和时区
- 币种、链、发送地址、接收地址
- 交易哈希与所属钱包
- 交易类型（例如普通转入、API 申请转出）
- 收支方向、原币金额和折算金额
- 排序、筛选、分页和交易详情

M2 Wallet 对照：

- 首期统一收款、提现、归集和手续费为一张可检索交易流水视图
- 保留业务单号、链上哈希、钱包、方向、来源、状态和确认数

### 6. 风控

Cregis 当前桌面端展示的能力：

- 策略列表、启停、权重、策略类型和触发规则
- 执行动作、签名动作和指定签名人
- 自动化任务，包含自动签名
- AML 地址风险查询、交易监控和资金来源追踪入口
- 策略、自动化、AML、团队 API 操作日志
- 地址白名单与黑名单
- 白名单支持币种、别名和地址；黑名单支持别名和地址

M2 Wallet 对照：

- Demo 已有地址格式校验、审批状态机和签名边界
- 首期补充单笔限额、日限额、黑白名单、暂停开关和规则命中原因
- AML 属于后续外部服务适配，不阻塞内部 Demo

### 7. 管理

Cregis 当前桌面端展示的能力：

- 团队订阅、功能额度与扩容
- 团队账户及费用明细
- 订单管理
- 成员管理
- 超级管理员、管理员、普通成员、云端审核等角色
- 团队安全：全局白名单校验、策略引擎、邀请成员
- 团队 API：名称、Key、权限、创建人、时间、状态和操作

M2 Wallet 对照：

- 首期实现管理员、财务审批、开发接入三个最小角色
- 首期实现 API Key 哈希存储、权限范围、启停和操作审计
- 订阅、计费和扩容不属于内部 Demo 范围

### 8. 应用市场

Cregis 当前桌面端展示稳定币汇款/跨境结算类扩展应用及应用管理入口。

M2 Wallet 对照：内部版本不做应用市场，保留适配器接口即可。
